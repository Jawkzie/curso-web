AUDIO Y VIDEO

En este apartado de esta pagina se estaran viendo las etiquetas y los atributos de <video> y <audio> y tambien sus atributos.

Atributos

  • <autoplay> funciona para que el navegador cargue el video automaticamente, pero no por completo, para eso se usa muted.
  • <muted> se utiliza junto al autoplay para que el navegador lo reproduzca en su totalidad y el usuario le suba el volumen.
  • <loop> por si se quiere reproducir el video en bucle.
  • <controls> para mostrar controles basicos en pantalla.
  • <poster> para poner una imagen de previsualizacion o miniatura.

LAZY LOADING

El lazy loading es un atributo que se le pone a los elementos que usualmente deben cargar, pueden ser tipo media, por ejemplo (imagenes / videos / audios) y este funciona para que los archivos carguen cuando el usuario los este viendo, mientras que si no se ven en pantalla no se carguen, funciona para una mejor optimizacion de la pagina, para que cargue mas rapido. Aunque este atributo tiene algunos errores y puede no mostrarle los elementos a los usuarios. SOLO FUNCIONA EN ETIQUETAS <img>
